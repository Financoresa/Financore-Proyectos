Se detalla la informacion del manejo de las carpetas

Documentacion.- Contiene documentos referentes a la base de datos Ej: Modelos de Datos, Detalles de Tablas
Respaldos.- Contiene los respaldos periodicos que se le realizaran a la base de datos con la finalidad de poder realizar reestauraciones de informacion en caso de ser necesario.
Script.- Contiene los Script de creacion de tablas, Bases de Datos, Consultas a la Base de datos complejas, etc.

Cabe indicar que cada uno de los archivos debe tener nombre unico en caso de ser una actualizacion se no se reemplazara, en dichos casos se renombrara el archivo y se pondra la fecha con la que se lo esta actualizando. 

Ej:
	scr_archivo.sql --> NUEVO
	scr_archivo_23012018.sql  --> ANTERIOR
